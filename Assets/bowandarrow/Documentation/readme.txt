Bow and Arrow - 2D Archery is a fun and challenging 2D archery game.
--------------------------------------------------------------------

Shoot for the bullseye on your Desktop, tablet or android phone
- bow and arrow
- sound effects
- collision detection via contact points
- changing distances
- correct physics implementation
- persistent score saving
- uses unity 4.6 gui engine
- fully commented c# source code


Just start the bowandarrow scene. <br>
